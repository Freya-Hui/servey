# 直接打开就有视频：最终版

这个版本用于：

- 本地直接打开 `index.html` 就能看到视频播放器
- GitHub 上传后，Vercel 也能读取视频文件

文件结构必须保持：

```text
index.html
media/
vercel.json
请上传这个版本.md
```

不要只上传 `index.html`，必须同时上传整个 `media` 文件夹。

视频路径现在是相对路径：

```text
media/C_part_video_materials/...
```

所以本地打开和线上部署都更稳。

这个版本保留：

- C 部分内嵌视频
- E 部分排序上移/下移按钮
- G 部分 AI 润色功能
