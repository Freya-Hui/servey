# 上传说明

请把这个文件夹里面的内容上传到 GitHub 仓库根目录。

必须上传：

- `index.html`
- `package.json`
- `vercel.json`
- `media` 文件夹

不要上传：

- `.zip` 压缩包
- `dist` 文件夹
- `node_modules` 文件夹

如果使用 GitHub 网页上传，请进入仓库后选择 `Add file` -> `Upload files`，然后把这个文件夹里的文件和 `media` 文件夹拖进去。

Vercel 设置：

- Framework Preset: `Other`
- Build Command: `npm run build`
- Output Directory: `dist`
