# 直接可用版本

请把这个文件夹里的全部内容上传到 GitHub 仓库根目录。

GitHub 仓库首页应该直接看到：

```text
index.html
package.json
vite.config.js
vercel.json
public
请上传这个文件夹里的全部内容.md
```

Vercel 会自动识别 Vite 项目。

Vercel 设置建议：

- Framework Preset：Vite
- Build Command：`npm run build`
- Output Directory：`dist`
- Install Command：`npm install`
- Root Directory：如果这些文件在仓库最外层，就留空

如果你把整个文件夹作为子文件夹上传，例如：

```text
repo/
  【直接可用_Vercel自动识别版】/
    index.html
    package.json
```

那么 Vercel 的 Root Directory 必须填写：

```text
【直接可用_Vercel自动识别版】
```

最推荐：不要套子文件夹，直接让 GitHub 仓库根目录显示 `index.html` 和 `package.json`。
